    \sectionheader{Abstract}
    Climate change poses unprecedented challenges to biological systems. We investigated adaptive mechanisms using multi-scale experimental designs, revealing \textbf{significant adaptive capacity} with temperature tolerance increasing by \textcolor{unibasRed}{\textbf{15--25\%}} under controlled conditions.
    
    \vspace{0.5em}
    \textcolor{unibasGray2}{\textbf{Keywords:}} climate adaptation, temperature tolerance, growth dynamics, thermal stress

    \vspace{1.2em}
    \sectionheader{Research Objectives}
    % Use basic itemize without enumitem
    \begin{itemize}
      \item Quantify adaptive responses to thermal stress across species
      \item Develop predictive models for climate resilience assessment
      \item Bridge the gap between laboratory and field studies
      \item Establish protocols for conservation applications
    \end{itemize}

    \vspace{1.2em}
    \sectionheader{Methodology}
    % Use basic itemize without enumitem
    \begin{itemize}
      \item \textbf{Design:} Multi-factorial (Temperature $\times$ Time $\times$ Species)
      \item \textbf{Sample size:} $n=120$ per treatment group
      \item \textbf{Duration:} 6-month controlled experiment
      \item \textbf{Analysis:} Mixed-effects models, Bayesian inference
      \item \textbf{Validation:} Cross-validation with field data
    \end{itemize}
  \end{column}
